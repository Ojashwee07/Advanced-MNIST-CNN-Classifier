# Advanced MNIST CNN Classifier

## Overview

This project builds an advanced CNN model for handwritten digit recognition using the MNIST dataset.

## Features

* Batch Normalization
* Dropout Regularization
* Multiple CNN Layers
* Improved Generalization
* High Accuracy Classification

## Technologies Used

* Python
* TensorFlow
* Keras
* NumPy

## Dataset

MNIST Handwritten Digit Dataset

## Model Architecture

* Conv2D (32)
* BatchNormalization
* MaxPooling2D
* Dropout
* Conv2D (64)
* BatchNormalization
* MaxPooling2D
* Dropout
* Dense (128)
* Dense (10 Softmax)

## Results

Achieved over 99% test accuracy with reduced overfitting.
